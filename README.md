# excel_to_word

## 介绍

excel_to_word 用于将 excel 表格中的每一个记录都转成 word 文档

使用效果如下：

excel 文件:

![excel 示例](image/excel_file.png)

第一条记录生成的 word 文件:

![word 示例](image/word_file.png)

## 安装

```bash
pip install -r requirements.txt
```

## 使用

```bash
python main.py -i excel_path -o word_dir
```
